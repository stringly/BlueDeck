UpdateComponentAndSetLineup(Component)

Adds a new Component or Updates an existing Component and adjusts the LineupPosition of all sibling Components within the ParentComponent

Declaration
public void UpdateComponentAndSetLineup(Component c)
Parameters
Type Name Description
Component c

The Component being added or edited.
Remarks

If this method is used to edit an existing component, then the Component parameter must be a new Component instance set to the form values of the existing component. This is because the method needs to know the "unedited" value of the Component's LineupPosition to determine whether the Component has been moved up or down in it's ParentComponent's Lineup. ///
