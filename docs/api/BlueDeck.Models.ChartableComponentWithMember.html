Remarks

This is not a direct representation of a Component entity. A GetOrgChart cannot render a component with multiple positions or multiple members. This class is used to map GetOrgChart Components, NOT domain Components. When rendering Member information into GetOrgChart, an instance of this class will be generated for each.
